# OIBSIP Web Development Internship - Level 2 Tasks

This repository contains the projects completed as part of the **Oasis Infobyte Web Development Internship**.  
It includes **Task 1: Calculator** and **Task 2: Tribute Page**.

---

## Task 1 – Calculator

### Project Title
Simple Web Calculator

### Objective
To design a simple calculator that performs basic arithmetic operations using HTML, CSS, and JavaScript.

### Technologies Used
- HTML
- CSS
- JavaScript

### Description
This project is a web-based calculator that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.  
The interface is simple, user-friendly, and responsive.

### Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear display function

## Task 2 – Tribute Page

### Project Title
Tribute Page for Dr. A. P. J. Abdul Kalam

### Objective
To create a tribute webpage dedicated to **Dr. A. P. J. Abdul Kalam**, highlighting his life, achievements, and inspirational journey.

### Technologies Used
- HTML
- CSS

### Description
This project is a simple tribute webpage honoring **Dr. A. P. J. Abdul Kalam**, the former President of India and a renowned scientist.  
It showcases his life, contributions, and achievements in a clean and readable layout.

### Features
- Biography section
- Achievements
- Clean webpage design
- Responsive layout
